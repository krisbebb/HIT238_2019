What are mobile apps?

Mobile Apps

  • Applications typically downloadable from an app store.
  • Platform specific, having access to specific system resources such as GPS or camera.
  • Also known as native mobile apps, examples include Facebook Messenger, Instagram, Snapchat.

Web Apps

  • Similar functionality and Ux as mobile apps, but accessed via the browser.
  • Not downloadable, and not specific to a particular platform.
  • Require an active internet connection.

Progressive Web Apps

  • Web Applications with offline storage allowing them to function like native mobile apps. Also known as PWAs.
  • Can function without an internet connection using cached data and access certain system resources.
  • Made possible with advancements in browser features.

What makes them mobile?

  • Responsive design allows applications to be usable across several platforms.
  • Designing for smaller screen size, swipe controls and using smartphone / smartwatch system resources makes an application mobile.
